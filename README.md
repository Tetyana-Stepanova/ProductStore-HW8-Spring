# ProductStore-HW8-Spring
SpringBott App,which makes it possible to work with the database

Данное приложение позволяет выполнять регистрацию пользоваталей администратором(пользователем с ролью ADMIN) и авторизацию пользователей, а также 
выполнять CRUD операции над следующими объектами:
==========================

Продукт:

Идентификатор UUID;

Наименование продукта;

цена;

Производитель;

==========================

производитель

Идентификатор UUID;

Наименование производителя;

==========================

Роль

Идентификатор UUID;

Название ROLE_USER или ROLE_ADMIN;

==========================

пользователь

Идентификатор UUID;

String email;

Строковый пароль;

Строка firstName;

String lastName;

Роли <Role>;

==========================

Роли пользователей:

1. Admin - имеет возможность выполнять все операции CRUD

2. Пользователь - имеет доступ только для чтения


В базе данных хранятся данные о производителях (Производитель) и товарах (Продукт), а также данные о пользователях, доступные только администратору(ROLE_ADMIN)

Каждый товар имеет одного производителя, а каждый производитель имеет набор товаров.


Страницы:

1. Manufacturers (список производителей для всех пользователей с ролью ROLE_USER + возможность создавать новых, редактировать и удалять созданных
производителей для ADMIN)

2. Products (список товаров для всех пользователей с ролью ROLE_USER+ возможность создавать новых, редактировать и удалять созданные товары для ADMIN)

3. Users (список всех ползователей приложения + возможность создавать новых, редактировать и удалять созданных пользователей) - ТОЛЬКО для ADMIN 
(и чтение и редактирование)

При создании товара выбор производителя реализовать с помощью выпадающего меню.
------------------------------------------------------------------------------------------------
------------------------------------------------------------------------------------------------
Перед запуском программы можно поменять настройки для своей базы данных в application.properties:
spring.datasource.username= Username of your connection
spring.datasource.password=Your Password
spring.datasource.jdbc-url=jdbc:mysql://Hostname:Port/name of yor empty DB
spring.datasource.hikari.jdbc-url=jdbc:mysql://Hostname:Port/name of yor empty DB

После запуска программа самостоятельно создаст все таблицы в базе данных и заполнит их

Вход в приложение через браузер по адресу http://localhost:8080

В базе данных уже будут 2 пользователя, для авторизации используйте email и password:
admin: admin@gmail.com( это логин для авторизации) admin (пароль для авторизации) - права администратора
user: user@gmail.com( это логин для авторизации) user (пароль для авторизации) - только чтение
